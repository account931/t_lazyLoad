# t_lazyLoad plugin.
#Uses lazyloadxt.extra.js to postpone Youtube video loading and boosts page loading speed.
# Uses Bootstrap templates with multiple effects.


To use lazy loading:
1. add <script src="js/lazyLoad/lazyloadxt.extra.js"></script> 
2. change in your video <iframe> all "scr" to "data-src"

Documentation: https://github.com/ressio/lazy-load-xt
